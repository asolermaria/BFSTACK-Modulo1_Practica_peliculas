Proyecto de práctica. Esta aplicación web permite añadir y explorar películas, realizar búsquedas y ver detalles de cada una, usando **HTML, CSS y JavaScript**.

## 📝 Descripción

Esta aplicación permite:

- Añadir películas
- Visualizar una lista de películas.
- Buscar películas por título.
- Ver detalles de cada película.

El objetivo es practicar conceptos fundamentales de **desarrollo front-end**, incluyendo estructura HTML, estilos CSS y lógica interactiva con JavaScript.
